## O que foi feito

Implementei um algoritmo de **busca binária** para encontrar contatos em uma agenda telefônica.

A agenda já vem com 10 contatos cadastrados e ordenados em ordem alfabética — que é a pré-condição para a busca binária funcionar. O programa pede o nome da pessoa e diz se ela foi encontrada ou não, mostrando o telefone caso sim.

---

## Como compilar

Com o terminal aberto na pasta do projeto, rode:

```
gcc -Wall -std=c99 -o agenda.exe main.c agenda.c
```

## Como executar

```
.\agenda.exe
```

Digite o nome exatamente como está na lista, por exemplo: `Carlos Lima`
